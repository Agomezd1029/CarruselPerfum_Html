# 🇨🇴 Carrusel Web - Fútbol Colombiano

Este proyecto es un carrusel de imágenes desarrollado con HTML y CSS puro, enfocado en representar la pasión del fútbol colombiano.

## 🚀 Características

- Animación automática con CSS (@keyframes)
- Diseño responsive (adaptable a celular)
- Imágenes en pantalla completa
- Overlay oscuro para mejorar legibilidad
- Texto centrado con efecto blur moderno

## 🛠️ Tecnologías usadas

- HTML5
- CSS3 (Flexbox + Animaciones)

## 🎯 Objetivo

Practicar la creación de interfaces visuales atractivas sin usar JavaScript, mejorando habilidades en:

- Maquetación
- Animaciones CSS
- Diseño responsive

## 📸 Contenido

El carrusel incluye imágenes relacionadas con:

- Fútbol
- Estadios
- Cultura deportiva

## 💡 Mejoras futuras

- Agregar botones manuales (JS)
- Indicadores de slide
- Transiciones más avanzadas
- Integración con API de deportes
